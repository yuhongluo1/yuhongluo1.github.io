📖 微信读书同步设置

跟着这几步做，只要做一次。

1️⃣ 打开微信读书网页版

打开 weread.qq.com → 点击"登录" → 用微信扫码登录

2️⃣ 打开开发者工具

按 F12 或 Cmd+Option+I

顶部点 Network（网络）

然后刷新页面 Cmd+R

3️⃣ 找到 Cookie

在 Network 列表里点第一个请求（一般是 booklist 或 user）

往下翻找到 Request Headers → Cookie

右键 → Copy Value

4️⃣ 把 Cookie 发给我

粘贴到聊天框里发给我就行！

Cookie 会安全存在你的电脑上，不会泄露。

搞完之后，你在微信读书上看书、划线、写笔记 → 每30分钟自动同步到你的网站 ✅
